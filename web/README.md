# AgentArena Anime Web GUI

Bright anime-inspired Web GUI prototype for AgentArena Local.

## Run

```powershell
cd web
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

## Build

```powershell
cd web
npm run build
```

## Lightswind

This UI imports `lightswind/styles.css` and follows the Lightswind copy-paste
component workflow documented in `../docs/lightswind-anime-web-gui.md`.
